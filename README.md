# Budget Lite

Budget Lite is a React personal finance planner for tracking income, expenses, category limits, and exportable money records.

## Features

- Add income and expense entries with amount, category, date, and notes
- Save all budget data in local storage
- Track income, expenses, balance, savings rate, and top spending category
- Set monthly category budget limits
- View budget progress bars by category
- Search and filter transactions
- Export full budget data as JSON
- Import a saved BudgetLite JSON file
- Export transactions as CSV
- Responsive dashboard layout for desktop and mobile

## Tech Stack

- React
- Vite
- CSS

## Getting Started

```bash
npm install
npm run dev
```
