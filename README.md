# Budget Lite

Budget Lite is a React personal finance dashboard for tracking income, expenses, spending categories, and monthly budget limits. I built it as a portfolio project to practice working with real user data, calculated totals, and a clean dashboard-style interface.

## Features

- Add income and expense entries with amount, category, date, and notes
- Track income, expenses, current balance, savings rate, and top spending category
- Set monthly budget limits for different spending categories
- View budget progress bars for each category
- Search transactions by name, category, or note
- Filter records by income, expense, or category
- Save budget data automatically in the browser with `localStorage`
- Export the full budget as JSON
- Import a saved Budget Lite JSON file
- Export transactions as CSV for spreadsheet use
- Responsive layout for desktop and mobile screens

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- Local Storage
- Browser File APIs

## How to Run Locally

```bash
npm install
npm run dev
```

To create a production build:

```bash
npm run build
```
