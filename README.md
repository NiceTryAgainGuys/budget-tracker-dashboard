# Budget Lite

Budget Lite is a React personal finance dashboard for tracking income, expenses, spending categories, and monthly budget limits. It is built to feel like a practical money planner instead of a basic calculator.

## Live Demo

Live link coming soon.

## Why This Project

This project turns a simple expense tracker into a more complete budgeting tool. It includes transaction management, budget progress, search, filters, exportable records, and a clean dashboard layout that helps users understand where their money is going.

## Key Features

- Add income and expense entries with amount, category, date, and notes
- Track income, expenses, balance, savings rate, and top spending category
- Set monthly category budget limits
- View progress bars for each spending category
- Search transactions by name, category, or note
- Filter records by income, expense, or category
- Save budget data automatically with `localStorage`
- Export the full budget as JSON
- Import a saved Budget Lite JSON file
- Export transactions as CSV for spreadsheet use
- Responsive layout for desktop and mobile screens

## User Workflow

1. Add income and expenses as they happen.
2. Set category limits for areas like food, transportation, school, or entertainment.
3. Review dashboard stats to understand spending habits.
4. Export records when the user wants to save or analyze their budget elsewhere.
